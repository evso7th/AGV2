Отлично, вот компактный, но конкретный чек‑лист для следующего 10–12‑мин рендера (Melancholic Drift, 120 тактов). Он расставляет «где и что включать/выключать», куда сажать мини‑мотив, когда разрешать sparkles/SFX, как ограничить ревивалы и микромутации. Удобно пройтись сверху вниз и отметить галочками.


1. TopVoice Manager для мелодии (мини‑мотив)
Сформируй 2‑тактовый мотив (например: 9 → 11 → 9 → 5 ступень) и вариируй его на границах бандлов (инверсия, последовательность, «ответ» на октаву). Дай мелодии паузы‑дыхания — аккомп держит фон.

Структура и секции
- [ ] Разметка частей: Intro 0–23, Build 24–53, Main 54–95, Release 96–119, Promenade 4 бара между сьютами.
- [ ] Бар‑лог на каждый такт: [Bar X] sec, techniques(bass/hats/ride/accomp/lead/arp), events счётчики.

Удары/перкуссия/верх
- Intro (0–23)
  - [ ] Kick 4/4 с бар 9 (или позже), очень тихо.
  - [ ] Hats_Closed16 (ghost, велосити 35–55), без open‑hat.
  - [ ] Ride: OFF.
  - [ ] Sparkles/SFX: OFF (жёстко).
- Build (24–53)
  - [ ] Hats_Closed16 (дыхание), open‑hat offbeat тихо.
  - [ ] Ride: OFF (до пика).
  - [ ] Том‑fill на 24 (переход в Build), ещё один по ситуации на 54.
- Main (54–95)
  - [ ] Ride окна:
    - [ ] ON 60–72 (штрихи на 2 и 4, велосити 50–60)
    - [ ] OFF 72–76 (окно отдыха ВЧ)
    - [ ] ON 81–92
    - [ ] OFF 92–95 (подготовка к Release)
  - [ ] Hats продолжают «дышать», open‑hat offbeat умеренно.
- Release (96–119)
  - [ ] Ride: OFF.
  - [ ] Hats — тише (−3 dB), open‑hat изредка/короче.
  - [ ] Том‑fill на 111–112 (перед затуханием).
- Promenade (4 бара)
  - [ ] Только технич. бридж: том‑ролл + тихий downlifter; без sparkles.

Бас и «ревивалы»
- Общие
  - [ ] Паттерн держим 8–16 тактов; вариация каждые 4–8 (грув «дышит»).
  - [ ] BassRevival кулдаун ≥ 16 тактов, бюджет на часть ≤ 2; логировать reason.
- Разрешённые точки ревивала
  - [ ] Бар 24 (вход в Build)
  - [ ] Бар 54 (вход в Main)
  - [ ] Бар 75 (межбандловая микро‑смена в Main)
  - [ ] Бар 96 (вход в Release)
- [ ] Во всех прочих барах — не ревивать, даже при желании двигателя.

Микромутации и «скоуп»
- Аккомпанемент (каждый такт, но с лимитом)
  - [ ] CAP: не более 2 микродействий/такт: Rhythm drift ИЛИ Velocity accent; Register drift — не чаще 1 раза в 2–4 такта (лучше на 4n).
- Бас
  - [ ] Микро — только на границах бандлов (каждые 8 тактов) и в разрешённые ревивальные точки.
- Бриджи
  - [ ] Только: pivot/step b7→1/fifth + том‑филл; никаких прочих микромутаций в эти 2–4 бара.

Мелодия (TopVoice Manager)
- Мини‑мотив (2 такта): 9 → 11 → 9 → 5 (относительно тонцентра D Dorian: E → G → E → A), легато, редкие паузы.
- Окна мотивов (выбираем 2–3 из списка):
  - [ ] 33–36 (ранний под’ём в Build)
  - [ ] 41–44 (перед входом ride)
  - [ ] 57–60 (на пике A)
  - [ ] 83–86 (пик B)
- Правила
  - [ ] Не унисон с верхним голосом аккомпа в момент атаки (микро‑сдвиг 20–40 мс).
  - [ ] Шаги преимущественно ±2 (соседи) и терции; редкая кварта; регистр ограничить (не выше A5 в меланхолии).
  - [ ] В окнах отдыха ВЧ (72–76) мелодия OFF или очень тихо.

Sparkles / SFX / Weather (гигиена)
- Запрет и бюджеты
  - [ ] Intro и Promenade: Sparkles = OFF; SFX = OFF.
  - [ ] В остальных частях: GAP ≥ 12 тактов; бюджет ≤ 2 на часть; сначала Sparkles, потом SFX (если нужно).
- Рекомендованные места
  - Build: [ ] Sparkles на 35, 42 (если GAP соблюдён).
  - Main: [ ] Sparkles на 62, 74, 89; [ ] SFX на 66 или 80 (один на часть).
  - Release: [ ] Sparkles на 108 или 112 (не громко); SFX — по вкусу, 1 раз.
- Weather events
  - [ ] Только в Build late/Main; GAP ≥ 16 тактов; бюджет ≤ 2/часть.

Пэды/гармония
- [ ] LPF падов: Intro 3.5→5 кГц; Build 5→6.5; Main 6.5–8; Release 6.5→4.5.
- [ ] «Окна тишины» ВЧ: в Main 72–76, в Release 104–108 — снять ride, снизить hats, LPF падов −1 кГц.

Микс/SC‑гардрейлы (быстро на слух/метрах)
- [ ] Sidechain GR по частям: Intro −2 dB, Build −2.5…−3 dB, Main −3…−3.5 dB, Release −2 dB.
- [ ] Onset rate (перкуссия+бас): Intro 0.3–0.6, Build/Main 0.6–1.0, Release 0.4–0.7 событий/такт.
- [ ] Активные sustain‑слои ≤ 3–4 (если >4 — sparsify аккомп или отключить один слой).
- [ ] Mid‑dip −1…−2 dB @ 250–400 Гц на сумме (против «стекловаты»).
- [ ] True peak ≤ −1 dBTP; LUFS pre‑master −10…−9 (меланхолия — оставить «воздух»).

Технические флажки (чтобы не словить прошлые артефакты)
- [ ] BassRevival: cooldown ≥ 16 тактов, budget ≤ 2/часть; лог reason=…; SKIP на остальных барах.
- [ ] Sparkles/SFX: блок в Intro/Promenade; GAP/бюджет включены.
- [ ] CAP микромутаций аккомпа: ≤ 2/такт; Register drift — не чаще 1 раза в 2–4 такта.
- [ ] Ride окна как выше; окно отдыха ВЧ обязательно (72–76 в Main).
- [ ] Бар‑лог включён (sec/tech/events), чтобы мы могли сверить постфактум.

Как слушать/сравнить (быстрый A/B)
- A: текущая версия (10 минут).
- B: версия по чек‑листу. При прослушивании отметь:
  - [ ] Усталость от ВЧ снизилась (за счёт ride‑окон)?
  - [ ] Мелодия обрела маленькую «дугу», а не только верх аккомпа?
  - [ ] Sparkles/SFX не «выпрыгивают» — только украшают переходы?
  - [ ] Бас стал стабильнее (реже «перезапускается» на слух)?
  - [ ] Переходы (24/54/96) читаются «ритуально», узнаваемо?

Если удобно, к следующему прогона присылай 30–40 баров бар‑логов по новой схеме (sec/tech/events + пометки, где сработал CAP/бюджет) — быстро сравним с эталоном и ещё докрутим (особенно места мотивов и окна отдыха ВЧ).
