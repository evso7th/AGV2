# Журнал Проекта "AuraGroove"

---

### ЗАПИСЬ: 2024-12-04 (План №159: "ShineOn Standard & Full Volume")

**ПРОБЛЕМА:**
Гитара `blackAcoustic` звучала слишком сухо и не соответствовала качеству оркестровки ансамбля.

**ПРИЧИНА:**
1. Использование сэмплированного инструмента вместо моделируемого в блюзовых сюитах.
2. Недостаточный дефолтный уровень громкости для солирующей партии.

**РЕШЕНИЕ:**
1. **Model Swap**: Во всех блюзовых блюпринтах (`winter.ts`, `dark.ts`) инструмент `blackAcoustic` заменен на `guitar_shineOn`.
2. **Standard Alignment**: `guitar_shineOn` теперь является основным сольным стандартом для блюза.
3. **Volume Maximization**: Дефолтная громкость мелодии в `useAuraGroove` поднята до 1.0.

**ВЫВОД:**
Достигнуто профессиональное, "гилморовское" звучание соло с богатыми обертонами и полным динамическим диапазоном.

---

### ЗАПИСЬ: 2024-12-04 (План №157: "Solo-First Orchestration & Budget Floor")

**ПРОБЛЕМА:**
В интро были слышны только бас и ударные. Соло-гитара Alvin Lee игнорировалась движком.

**ПРИЧИНА:**
1. **Priority Inversion**: Бас и Ударные забирали энергию первыми. На малых уровнях напряжения (tension < 0.2) остатка бюджета не хватало для Solo (цена 50).
2. **Tight Budget**: Базовый бюджет 60 был слишком мал для одновременного звучания Solo + Full Drums.

**РЕШЕНИЕ:**
1. **Solo-First**: В методе `generateBar` соло теперь получает бюджет первым. Оно гарантированно играет, если разрешено навигатором.
2. **Budget Floor**: Базовый бюджет поднят до 80. Теперь трио (Solo+Bass+Drums) всегда умещается в лимит.
3. **Price Re-calibration**: Цены на второстепенные партии снижены (Bass Pedal: 5, Picking: 10).
4. **Volume Adjustment**: Вес аккомпанемента поднят до 0.15 для лучшей слышимости фона.

**ВЫВОД:**
Иерархия "Соло - Главное блюдо" теперь защищена на уровне оркестрового бюджета. Гитара Alvin Lee доминирует в миксе с первого такта.

### ЗАПИСЬ: 2024-12-04 (План №156: "Narrative Vitality & Bass Floor")

**ПРОБЛЕМА:**
1. Интро было слишком пустым из-за нехватки энергетического бюджета для соло.
2. Выбирались технические "Outro" планы как основной сюжет.
3. Бас уходил в инфразвук (ниже 20 Гц) при низких тониках.
4. В карте сэмплов гитарных аккордов присутствовали нежелательные Telecaster сэмплы.

**ПРИЧИНА:**
1. Порог базового бюджета (40) был ниже стоимости соло (50).
2. Отсутствие фильтрации планов при выборе повествовательной линии.
3. Прямое вычитание октавы без проверки слышимого диапазона.
4. Избыточные записи в `samples.ts`.

**РЕШЕНИЕ:**
1. **Budget Floor**: Базовый бюджет поднят до 60. Теперь соло гарантированно вступает с 0-го такта.
2. **Story Filter**: Из пула выбора планов исключены все ID, содержащие "OUTRO".
3. **Sub-bass Protection**: Внедрен порог MIDI 24. Все ноты ниже автоматически транспонируются вверх.
4. **Sample Cleanup**: Из `ACOUSTIC_GUITAR_CHORD_SAMPLES` удалены все ссылки на Telecaster.

**ВЫВОД:**
Ансамбль обрел "волю к жизни" в интро, бас стал физически слышимым, а библиотека сэмплов очищена.
