
# Журнал Проекта "AuraGroove"

---

### ЗАПИСЬ: 2024-12-04 (План №260: «ENSEMBLE ANCHOR & PIANO X-RAY»)

**ПРОБЛЕМА:**
1. Инструменты внезапно исчезали из микса в Dark Blues (т.н. «обмороки»), так как активация проверялась каждый такт заново.
2. Пианист продолжал зацикливаться, а Страж не давал достаточно данных для анализа «врага» (нотной последовательности).

**РЕШЕНИЕ:**
1. **Ensemble Anchor**: Внедрена память `activatedParts`. Если музыкант один раз вступил в игру, он больше не исчезает до конца сюиты. Это «цементирует» состав ансамбля.
2. **Dynamic Morphing Safeguard**: Сохранена возможность смены тембра по энергии для закрепленных инструментов. Музыкант остается на сцене, но меняет «голос».
3. **Piano X-Ray Telemetry**: Добавлен лог `[SOR] Piano Sequence`, который печатает MIDI-ноты и тики пианиста. Это позволяет проводить «посмертный анализ» циклов.
4. **Heavy Vaccine**: Шаг смещения `pianoStagnationOffset` увеличен до 500-1000 единиц для гарантированного выхода из петли.

### ЗАПИСЬ: 2024-12-04 (План №259: «DYNAMIC ENSEMBLE REALIGNMENT»)

**ПРОБЛЕМА:**
Настроения "Dark" и "Anxious" теряли инструменты в середине пьесы из-за падения напряжения. Настроение "Anxious" было избыточно суетливым на 100 BPM.

**РЕШЕНИЕ:**
1. **Persistent Ensemble**: Инструменты больше не отключаются по бюджету. Вместо тишины при низком напряжении они переходят на "Ambient Fingerstyle" или простые арпеджио.
2. **Timbral Morphing (V3)**: Внедрена динамическая цепочка тембров: `blackAcoustic` (Tension < 0.4) -> `cs80` (0.4-0.7) -> `guitar_shineOn` (> 0.7).
3. **BPM Density Hygiene**: Реализована обратная зависимость плотности от темпа (`bpmFactor`). Чем выше BPM, тем больше пауз ("воздуха") вставляется в паттерны.
4. **Smoldering Dark**: Профиль напряжения Dark Blues изменен на "тлеющий" (0.6 - 0.9) для предотвращения энергетических провалов.
5. **Calmed Anxiety**: Сглажена волна напряжения в Anxious Blues, снижена базовая плотность нот для комфортного фонового прослушивания.
