
# Журнал Проекта "AuraGroove"

---

### МИЛЬСТОУН: 24-02-2026 (ПЛАН №761: BLUES ENSEMBLE FOCUS)
**ИТОГ**: Фокусировка диапазона аккомпанемента и тепловая модернизация органа.

**Ключевые Изменения**:
*   **Octave Constraint**: Все инструменты сопровождения в блюзе теперь жестко ограничены 3-4 октавой (48-71 MIDI). Это создает плотную, винтажную "середину".
*   **Organ Filter**: В движок органа добавлен честный LPF-фильтр. Теперь орган звучит мягко и не пронзительно.
*   **Sub-Bass Layer**: Для органа `soft_jazz` добавлен дополнительный саб-басовый слой, дающий глубину при игре одиночных нот.
*   **Range Normalization**: Принудительная фильтрация октав теперь работает для аккомпанемента, гармонии и пианиста-тени.
