# Генетическая Система Наследования AuraGroove (v1.1)

## 1. Введение: Теория Конвергенции
Основная гипотеза развития системы заключается в том, что по мере накопления "Шедевров" (резонансных состояний) в глобальном генофонде, частота возникновения новых шедевров должна возрастать. Это происходит благодаря отбору наиболее гармоничных музыкальных структур и их использованию в качестве "родителей" для новых сессий. По мере роста базы данных "правильных" решений, пространство музыкального поиска сужается вокруг резонансных зон.

## 2. Цикл Рождения Сюиты

### Шаг 1: Семантический Отбор (Birth Selection)
В момент инициализации СОР выбирает из оцифрованной библиотеки базовый мелодический "Лик". 
- Выбор фильтруется по текущему `Mood`. 
- Система запоминает последние выбранные лики, чтобы избежать повторов в коротком промежутке времени.

### Шаг 2: Первичная Трансформация (Birth Transformation)
Выбранный лик проходит через одну из четырех фрактальных трансформаций в момент "Рождения" (Epoch 0). Это гарантирует, что даже один и тот же лик может породить разные ветви развития:
- **Inversion**: Зеркальное отражение нот относительно первой ступени.
- **Retrograde**: Реверс временной последовательности нот.
- **Transposition**: Сдвиг в другую тональность внутри блюзовой сетки.
- **Rhythmic Jitter**: Микро-сдвиги времени старта нот.

### Шаг 3: Генетическое Скрещивание (DNA Crossover)
Если в глобальной памяти (`masterpieces`) есть удачные предки, система выбирает случайного "родителя" из последних 20 записей.
- **Bitwise Crossover**: Текущий случайный Seed скрещивается с Seed-ом предка на битовом уровне (маска `0x55555555`).
- Это позволяет новому "семени" наследовать структурные особенности (темп, гармоническую сетку, плотность), которые привели предка к успеху, сохраняя при этом новизну.

### Шаг 4: Посев (Sowing)
На основе финального скрещенного семени генерируются все структурные параметры сюиты:
- Гармонический скелет (цепь Маркова или блюзовая сетка).
- Энергетическая карта (`Tension Map`).
- Ритмический стиль и выбор инструментов.

## 3. The Legacy Bridge: Культурное Наследование
Проект «Legacy» внедряет в систему оцифрованный опыт великих блюзменов. Вместо математических абстракций, СОР может использовать в качестве «родителя» эталонную человеческую фразу.

- **Legacy Licks (L31-L180)**: Библиотека из 150+ фраз, дистиллированных из живого блюза.
- **Semantic Anchoring**: Лики помечены тегами (`slow-burn`, `texas-shuffle`, `jazzy`), что позволяет системе наследовать не только ноты, но и «характер» исполнения.
- **Hybrid Evolution**: Скрещивание человеческого лика с фрактальной мутацией рождает музыку, которая звучит «по-человечески» в своей основе, но развивается по неземным законам.

## 4. Мониторинг и Оценка (AI Arbitrator)
AI-Арбитр постоянно оценивает звучание каждого такта через Матрицу Резонанса. При достижении порога **0.78** (усредненная мера красоты), семя текущей сессии автоматически сохраняется в облако Firestore, становясь потенциальным "предком" для будущих поколений.
