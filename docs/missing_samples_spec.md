# Спецификация Недостающих Сэмплов: Акустическая Гитара

Этот документ определяет "идеальный" набор сэмплов для реалистичной имитации акустической гитары. Цель — иметь как минимум 2-3 техники звукоизвлечения для каждой из шести открытых струн.

**Формат имени файла (рекомендуемый):** `acoustic_guitar_[NOTE]_[TECHNIQUE].wav` (например, `acoustic_guitar_E2_pick.wav`)

## Дополнительные Сэмплы (для реализма)

- **Slides:** Несколько сэмплов скольжения пальцев по струнам разной длительности.
- **Mutes:** Сэмплы приглушенных струн.
- **Body Hits:** Сэмплы легких ударов по корпусу гитары.
