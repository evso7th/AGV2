# Спецификация Недостающих Сэмплов: Акустическая и Электрогитара (v2.0)

Этот документ определяет целевой набор сэмплов для достижения максимального реализма через гибридный синтез (Layering + Physical Modeling).

## 1. Транзиенты для Гибридного Синтеза (Attack Bursts)
*Нужны очень короткие файлы (30-70 мс), содержащие только момент удара.*

| Тип импульса | Описание | Инструмент |
|:--------------|:---------|:-----------|
| **pick_hard** | Резкий удар медиатором (пластик) | Telecaster / Lead |
| **pick_soft** | Мягкое касание медиатором | Rhythm / Clean |
| **finger_flesh**| Удар подушечкой пальца (мягкий) | Black Acoustic |
| **finger_nail** | Щипок ногтем (яркий) | Solo Acoustic |
| **string_slap** | Удар большим пальцем по струнам | Bass / Percussive Gtr |

## 2. Шумовые Артефакты (Humanization Layer)
*Файлы для подмешивания в паузах или между нотами.*

| Тип шума | Описание | Назначение |
|:----------|:---------|:-----------|
| **fret_squeak**| Скрип пальцев при смене позиции | Реализм переходов |
| **string_buzz**| Легкий дребезг струны о лад | Грязный блюз / Характер |
| **body_thump** | Глухой удар по деке гитары | Перкуссионный аккомпанемент |
| **pick_scrape**| Скольжение медиатора по обмотке | Агрессивные сольные лики |
