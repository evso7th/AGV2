# Спецификация Недостающих Сэмплов: Акустическая и Электрогитара (v2.1)

Этот документ определяет приоритетный набор сэмплов для достижения максимального реализма через гибридный синтез (Layering + Physical Modeling).

## 1. Priority 1: High Impact Sounds (The "Soul" Noises)
*Эти звуки дают 90% эффекта «реальности». Нужны очень короткие импульсы (30-100 мс).*

| Тип импульса | Описание | Назначение |
|:--------------|:---------|:-----------|
| **pick_strike** | Резкий щелчок медиатора по металлу | Атака для Telecaster. Дает «пробиваемость» в миксе. |
| **flesh_pluck** | Мягкий, глухой удар подушечкой пальца | Атака для Black Acoustic. Дает теплоту и объем. |
| **fret_squeak** | Нетональный скрип пальцев по обмотке | Самый сильный маркер реальности при смене позиций. |
| **string_stop** | Звук прикосновения ладони к струне | Убирает стерильную тишину при noteOff. |
