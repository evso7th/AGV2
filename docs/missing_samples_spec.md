# Спецификация Недостающих Сэмплов: Акустическая и Электрогитара (v2.2)

Этот документ определяет приоритетный набор сэмплов для достижения максимального реализма через гибридный синтез (Layering + Physical Modeling).

## 1. Priority 1: High Impact Sounds (The "Soul" Noises)
*Эти звуки дают 90% эффекта «реальности». Нужны очень короткие импульсы (30-100 мс).*

| Тип импульса | Описание | Source Compatibility | Назначение |
|:--------------|:---------|:---------------------|:-----------|
| **pick_strike** | Резкий щелчок медиатора по металлу | **Electric/Steel Only** | Атака для Telecaster. Дает «пробиваемость». |
| **flesh_pluck** | Мягкий, глухой удар подушечкой пальца | **Acoustic/Bronze Only** | Атака для Black Acoustic. Дает теплоту. |
| **fret_squeak** | Нетональный скрип пальцев по обмотке | **Universal (Steel)** | Маркер реальности при смене позиций. |
| **string_stop** | Звук прикосновения ладони к струне | **Universal** | Убирает стерильную тишину при noteOff. |
