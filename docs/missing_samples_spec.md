
# Спецификация Недостающих Сэмплов: Акустическая и Электрогитара (v2.3)

Этот документ определяет приоритетный набор сэмплов для достижения максимального реализма через гибридный синтез (Layering + Physical Modeling).

## 1. Priority 1: High Impact Sounds (The "Soul" Noises)
*Эти звуки дают 90% эффекта «реальности». Нужны очень короткие импульсы (30-100 мс).*

| Тип импульса | Описание | Source Compatibility | Статус |
|:--------------|:---------|:---------------------|:-------|
| **pick_strike** | Резкий щелчок медиатора по металлу | **Electric/Steel Only** | `Done (Donor Mode)` |
| **flesh_pluck** | Мягкий, глухой удар подушечкой пальца | **Acoustic/Bronze Only** | `Done (Donor Mode)` |
| **fret_squeak** | Нетональный скрип пальцев по обмотке | **Universal (Steel)** | `Pending` |
| **string_stop** | Звук прикосновения ладони к струне | **Universal** | `Pending` |

## 2. Core 6: The Representativ-ы (Hybrid Donors)
*Вместо поиска "чистых" открытых струн, мы используем существующую библиотеку нот как доноров 20мс транзиентов.*

| Источник (Donor) | MIDI Диапазон | Роль | Статус |
|:--------------|:---------|:-------------|:-------|
| **Telecaster Clean** | E2 - C6 | Донор «металлического» щипка для электрогитары. | `Implemented` |
| **Black Acoustic** | E3 - E6 | Донор «деревянного» щипка для акустики. | `Available` |

## 3. Алгоритмическая Хирургия (Implemented in Samplers)
Для извлечения транзиента используется следующая логика:
1.  Выбор ближайшего записанного сэмпла к целевой MIDI-ноте.
2.  Ограничение времени звучания: `startTime + 0.02s` (20 мс).
3.  Микро-затухание (fade-out) последних 5 мс транзиента для предотвращения цифровых щелчков.
