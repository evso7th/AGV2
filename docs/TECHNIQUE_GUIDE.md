# Справочник Техник Исполнения (AuraGroove)

Этот документ является "паспортом" возможностей наших виртуальных музыкантов. Он описывает все техники игры, которые можно указывать в блюпринтах (`instrumentRules`) для управления характером исполнения.

---

## 1. Бас-гитарист (Bass)

Басист — основа грува. Его техники определяют ритмический "кач" и гармонический фундамент.

### Блюзовые и Роковые Техники

| Техника | Описание | Рекомендации по применению |
| :--- | :--- | :--- |
| **`riff`** | Играет заранее определенный, стилистически верный рифф. | Основа для блюзовых и роковых композиций. Выбирается движком автоматически. |
| **`walking`** | Строит плавную мелодическую линию ("шаги"), соединяя аккорды. | Идеально для джаза, свинга и медленного блюза, где нужно "вести" гармонию. |
| **`boogie`** | Классический повторяющийся паттерн "буги-вуги". | Для создания энергичного, танцевального грува в быстром блюзе и рок-н-ролле. |

### Эмбиент и Текстурные Техники

| Техника | Описание | Рекомендации по применению |
| :--- | :--- | :--- |
| **`long_notes`** | Играет длинные, протяжные ноты (обычно тонику аккорда). | Основа для эмбиента и транса. Создает плотный, гудящий фундамент. |
| **`drone`** | Разновидность `long_notes` с очень медленной атакой и затуханием. | Для создания почти неподвижного, медитативного звукового ландшафта. |
| **`pulse`** | Ритмичное повторение одной ноты (обычно 8-ми или 16-ми). | Основная техника для создания "кача" в трансе и электронике. |

### Синтезаторные Техники (Legacy)

| Техника | Описание | Рекомендации по применению |
| :--- | :--- | :--- |
| **`arpeggio`** | Быстрый перебор нот текущего аккорда. | Для электронных стилей, где нужен быстрый, секвенсорный бас. |
| **`portamento`** | Плавное "скольжение" высоты тона между нотами. | Для создания "жидкого", текучего звука синтезаторного баса. |

## 2. Аккомпанемент (Accompaniment)

Партия аккомпанемента отвечает за гармоническую и ритмическую текстуру.

| Техника | Описание | Рекомендации по применению |
| :--- | :--- | :--- |
| **`long-chords`** | Длинные, слитные аккорды, создающие гармоническую "подушку". | **Основа для Эмбиента.** Также используется в балладах и интро. |
| **`rhythmic-comp`** | Короткие, синкопированные "удары" аккордами. | Идеально для фанка, соула, энергичного блюза. Создает грув. |
| **`power-chords`** | Играются только тоника и квинта аккорда. | Мощный, "тяжелый" звук для рока, метала и индастриала. |
| **`alberti-bass`** | Классическая фортепианная фигурация (низ-верх-середина-верх). | Придает музыке "классическое", структурированное звучание. |
| **`arpeggio-slow`** | Медленный перебор нот аккорда. | Создает ощущение полета, используется в трансе и мечтательных пассажах. |
| **`arpeggio-fast`** | Быстрый перебор нот аккорда. | Добавляет энергии и движения. |

## 3. Соло-гитарист (Melody - Guitar Licks)

Эти техники указываются в библиотеке блюзовых "ликов" (`blues_guitar_solo.ts`) и применяются движком для придания выразительности.

| Техника | Описание |
| :--- | :--- |
| **`sl` (slide)** | Плавный "подъезд" к ноте. |
| **`h/p` (hammer/pull)** | Легатная техника без атаки медиатором. |
| **`bn` (bend)** | Подтяжка струны для повышения высоты тона. |
| **`vb` (vibrato)** | Легкая пульсация высоты тона на длинных нотах. |
| **`gr` (grace note)** | Очень короткая нота-украшение перед основной. |
| **`ds` (double-stop)** | Одновременная игра двух нот. |

---

## 4. Ударные (Drums)

Хотя для ударных мы используем паттерны, а не техники в том же смысле, эти термины могут встречаться в описании событий.

| Техника | Описание |
| :--- | :--- |
| **`hit`** | Стандартный удар. |
| **`ghost`** | "Призрачная нота" — очень тихий, едва слышный удар для создания грува. |
| **`roll`** | Барабанная дробь, обычно на малом барабане или томах. |
| **`swell`** | Нарастающий по громкости удар по тарелке (крещендо). |
