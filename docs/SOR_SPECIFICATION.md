
# Техническая Спецификация: СОР (Система Обеспечения Разнообразия) v2.18

## 1. Концепция
СОР — это многоуровневая когнитивная надстройка над FractalMusicEngine, превращающая алгоритмическую генерацию в осмысленное музыкальное повествование. Система имитирует работу живого ансамбля, где каждый участник обладает памятью, слышит коллег и подчиняется драматическому сюжету.

---

## 2. Ключевые Технологии v2.18 (PLATINUM / STABLE)

### 2.1. Ensemble Symbiosis (Симбиоз Ансамбля)
*   **Level 1: Sibling Search**: Приоритетная "сшивка" басовых и мелодических фраз из одного и того же трека Наследия.
*   **Level 2: Rhythmic Mirroring**: Адаптация плотности баса под энтропию (сложность) соло-партии.
*   **Level 3: Dynamic Fills**: Вставка басовых проходок в моменты пауз ("вдохов") солиста.

### 2.2. Genetic Anchor Logic (Генетический Якорь)
*   **Mechanism**: Фиксация одного `compositionId` в качестве эксклюзивного донора аксиом на всю сюиту.
*   **Result**: Обеспечение тематической и авторской целостности композиции.

### 2.3. Genre Sovereignty (Жанровый Суверенитет)
*   **Mechanism**: Автоматический фильтр облачных аксиом по полю `genre` при неактивном Якоре.
*   **Result**: 100% стилистическая чистота Наследия. Треки Trance звучат только в трансе, Blues — только в блюзе.

### 2.4. Sonic Cube Chords
*   **Volume**: 242 оцифрованных аккорда.
*   **Algorithm**: Round Robin (выбор случайного дубля из массива) для предотвращения механистичности.

### 2.5. Headroom Protocol
*   **Logic**: Жесткая калибровка `VOICE_BALANCE` с запасом -6dB перед лимитером. Устраняет межканальные искажения.

## 3. Ревизия Функций по Уровням

### Уровень 4: СОЧИНЕНИЕ (The Mind of SOR)
*   **The Yoga of Radiance**: Трехуровневая модель Света (Epic/Enthusiastic/Joyful).
*   **Heritage Intelligence**: Автоматическая ротация ДНК с использованием Shuffle Bag и динамическим жанровым фильтром.
*   **Universal Texture Engine**: Использование `AmbientBrain` для всех не-блюзовых жанров, обеспечивающее поддержку облачного Наследия для Trance, Progressive и House.
