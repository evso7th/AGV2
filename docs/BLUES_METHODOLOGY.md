## 2. Физика Исполнения (Ensemble Sophistication)

### 2.1. Унисон 2.0 (The Obsidian Protocol)
*   **Strict Unison**: Бас и Аккомпанемент (Пад) дублируют друг друга в реальном времени.
*   **Harmonic Enrichment**: Пад использует случайные инверсии и добавляет "теневые" ступени (9, 11) для глубины.

### 2.2. Нарративное Пианино
*   **The Emotional Responder**: Пианист реагирует на динамику напряжения (Tension Momentum).
*   **Exhale Passages**: При падении напряжения (momentum < 0) пианист исполняет нежные восходящие пассажи.
*   **Syncopated Drops**: В моменты покоя используются редкие, непредсказуемые акценты по ступеням аккорда.

## 3. Нарративная Мелодика (The Melodic Glue)

### 3.1. Принцип Цепкого Легато
*   **Fluid Flow**: Если интервал между нотами меньше 1 тика, длительность первой ноты увеличивается на 0.15с, создавая физическое перекрытие звука.
*   **Hybrid Synthesis**: Каждая нота синтезатора сопровождается 20мс транзиентом реального инструмента (Telecaster) для "физического" укуса.
