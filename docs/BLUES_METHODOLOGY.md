## 2. Физика Исполнения (Performance Tiers)

Манера игры ансамбля жестко привязана к уровням Tension (Схема 20/60/20):

### 2.1. Tier 1: Интимность (T < 0.4)
*   **Бас**: Дроны (1 нота на 1-2 такта).
*   **Аккомпанемент**: Мягкие «подушки» (Long Chords).
*   **Пианино**: Редкие, интимные ответы по ступеням аккорда.

### 2.2. Tier 2: Диалог (0.4 <= T <= 0.7)
*   **Бас**: Упругий рифф (4-8 ударов на такт).
*   **Аккомпанемент**: Слышимые арпеджио на фоне падов.
*   **Пианино**: Простые арпеджио и лирические вставки.

### 2.3. Tier 3: Накал (T > 0.7)
*   **Бас**: Полноценный волкинг-бас (4 ноты на такт).
*   **Аккомпанемент**: Мощные аккордовые удары и вязкие пассажи («навал»).
*   **Пианино**: Экспрессивные арпеджио и быстрые блюзовые пассажи.

## 3. Контроль Разнообразия (Narrative SOR)

### 3.1. Единая Мутация
Каждые 12 тактов ансамбль меняет «состояние сознания». Тип мутации (inversion, jitter, rhythm) применяется ко всем инструментам синхронно.

### 3.2. Вакцинация Ансамбля (Stagnation Strikes)
Система отслеживает зацикливание через хеширование фраз Гитариста и Пианиста:
1.  **Strikes**: Если любой из солистов повторяет фразу, счетчик `stagnationStrikes.micro` растет.
2.  **Vaccine**: При достижении 3-х страйков система принудительно сбрасывает Аксиому для **всего** ансамбля.
3.  **Logs**: Логи `[Narrative]` теперь содержат точный `BarCount` для всех участников.

### 3.3. Сохранение Хвостов
Для инструментов с естественным затуханием (Фортепиано, Акустика) запрещено принудительное прерывание сэмпла (`source.stop`). Звук должен растворяться естественным образом.
