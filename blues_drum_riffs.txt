Формат понятный для движка: указываю tpb=12 (12 триольных восьмых на такт), где удар — tick (0…11). Для шапки: типичный «shuffle» хэт играет первый и третий триоль в каждом бите: ticks = [0,2, 3,5, 6,8, 9,11].

Общее
- tpb: 12 (триплет‑восьмые). В каждом бите три позиции: n*3, n*3+1, n*3+2; shuffle = «1» и «a»: (n*3) и (n*3+2).
- Обозначения: HH — closed hi‑hat, OH — open hat, SD — snare, K — kick, R — ride, T — tom. Пример массива: [0,2,3,5,6,8,9,11]
- Велосити (рекомендации): HH 55–70 (ghost 40–50), SD 78–88 (ghost 55–65), K 75–85, R 60–70, T 65–75.

Правила мутаций (общие и по настроениям)
Микро (каждый такт/через такт)
- Ghost‑ноты SD: добавить SD на [2] и/или [8] (vel −20 от SD основного).
- Open‑hat: OH на [11] раз в 2–4 такта (Joyful/Enthusiastic/Neutral), тише — в Calm/Dreamy (раз в 4–8).
- Kick‑вариант: заменить [6] → [5] (чуть «тянет» вперёд) или добавить [10] на подъём в бар 12.

Мезо (каждые 4–8 тактов/на границах фраз)
- Ride on/off: включить Ride (Joyful/Enthusiastic — на соло‑хорах; Melancholic/Dark — только в пике), затем выключить.
- Ghost хэты → полнотелый shuffle: Dreamy/Calm — чередовать «редкие» HH и полный HH_SHUFFLE.
- Тихий том на «and of 4»: T [11] или [10–11] в конце фразы.

Макро (секции/стоп‑тайм)
- Stop‑time (бар 9–10): выключить HH/R, оставить K+SD хиты — [0] и [6/9] (KICK_STOP/SNARE_STOP), лид держит ноту.
- Бриджи: 1‑барный TOM_FILL перед переходом (см. филлы ниже), затем возврат к исходному паттерну.

Библиотека филлов (готовые)
Полутактовые (6 тиков)
- Fill A (snare+tom триоль): SD [0,3], T [4,5] (или [10,11] в конце бара)
- Fill B (kick‑snare‑kick): K [0,4], SD [3], HH «закрыть» на [5]

Полнотактовые (12 тиков)
- TOM_FILL_EUCLID (ровный рассыпной): T [0,3,5,8,11], SD [9] (акцент возвращения)
- Shuffle Roll: SD ghost на [0,2,3,5,6,8,9,11] (vel 55–65), SD акцент [9], K [0,6] — роящий «ролл».
- Crash‑тапа: в конце секции — OH/Crash (если есть) на [0], затем тишина HH до [3], возвращение в основной бит.

Примеры применения филлов по настроениям
- Joyful/Enthusiastic: каждые 4–8 тактов Fill A или TOM_FILL_EUCLID; перед соло — Crash на начало хора.
- Neutral: 8–12 тактов между филлами, лучше короткие (Fill B).
- Dreamy/Calm: редко; лучше «ghost roll» на [8–11] с тихим SD/Т.
- Melancholic/Gloomy/Dark: 8–12 тактов; предпочтительнее tom‑филлы на [10–11] и хиты на стоп‑тайме.

Динамика/человечность
- Humanize тайминг: ±10–20 мс для HH/SD/Т, ±10 мс для K (осторожно).
- Велосити: варьируй HH (на сильные [0,6] чуть громче), ghost −20…−30 от основного.

Ниже — 40 блюзовых битов (по 5 на каждое настроение) для 12/8 (tpb=12), пригодные для прямого рендера. Я даю:

- Общую библиотеку масок (ticks 0..11) для HH/SD/K/R/T и филлов.
- 40 «рецептов»: для каждого бита — базовый набор масок, вариации, стоп‑тайм (бар 9–10), филл на бар 12.
- Темповые коридоры и динамику.

Библиотека масок (tpb=12)
- Хэты (HH)
  - HH_SHUFFLE = [0,2, 3,5, 6,8, 9,11]         // классический шаффл
  - HH_SPARSE  = [0,2, 6,8]                     // редкий хэт
  - HH_GHOST   = [0,2,3,5,6,8,9,11] (ghost на [2,5,8,11] — вел. 45–55)
  - OH_11      = [11]                           // открытый хэт на «а 4»
- Малый (SD)
  - SN_BACK    = [3,9]                          // бэкбит
  - SN_STOP    = [0,6]                          // стоп‑тайм (хиты на 1 и 3)
  - SN_GHOST   = [2,8]                          // призрачные
- Бочка (K)
  - K_BASE     = [0,6]                          // 1 и 3
  - K_DRIVE    = [0,6,8]                        // добавка на «а 3»
  - K_PUSH     = [0,5,6]                        // лёгкий «подъём»
  - K_SWAMP    = [0,8]                          // «грязноватый» свамп
  - K_MIN      = [0]                            // минимализм
  - K_STOP     = [0,6]                          // стоп‑тайм (с SD_STOP)
- Райд (R)
  - R_QTR      = [3,6,9]                        // четверти (триольное ощущение)
  - R_HALF     = [6,9]
- Тома (T)
  - T_11       = [11]
- Филлы (на бар 12, или конец фразы)
  - F_TOM_EUCL = T: [0,3,5,8,11], SD: [9]       // рассыпной том‑филл + акцент
  - F_SD_ROLL  = SD: [0,2,3,5,6,8,9,11], K:[0]  // «ролл» щадящий
  - F_KS_KICK  = K: [0,4], SD: [3]              // короткий KS на пол‑такта
Динамика (рекомендации)
- HH 55–70 (ghost 45–55), SD 78–88 (ghost −20), K 75–85, R 60–70, T 65–75.

Как читать «рецепт» бита
- base: какие маски на барах 1–8 и 11.
- variations: мелкие мутации (ghost, OH на [11], доп. бочка).
- bars9_10: стоп‑тайм (или альтернативный шаблон).
- bar12: филл.
- ride: где включать/выключать райд (если есть).
- tempo: целевой коридор (BPM).

Правила рендера (как «сложить» 12 тактов)
- Бары 1–8 и 11: ставь base‑маски (с вариациями по описанию).
- Бары 9–10: стоп‑тайм (SN_STOP+K_STOP; HH/R off), если указано.
- Бар 12: добавь выбранный филл (F_TOM_EUCL / F_SD_ROLL / F_KS_KICK) и возвращайся в base в следующем хоре.
- Ghost‑ноты и OH: по вариациям — не чаще 1–2 раз в 4 такта.
